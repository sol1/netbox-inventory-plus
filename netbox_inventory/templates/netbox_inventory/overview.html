Welcome to NetBox Inventory Plus

Introduction
The overview is intended to be a guide to the intended workflows when using Inventory Plus. But you aren't required to use all objects to manage your Assets.
The bare minimum required is the Asset object, everything else is optional.

Full Asset Creation

  • Add Supplier
  • Add Bill of Materials (BOM)
  • Create Assets from new BOM So the Assets are associated with the BOM
  • Add Purchase
  • Assign Existing Assets with Purchase
  • Add Delivery
  • Assign Existing Assets with Deliver
  • Scan Serial Numbers in Bulk
  • Deploying Assets to Devices and Modules
Transfering Assets
Inventory Plus allows you to track the transfer of assets between location. This lets you know where your assets including with a courier in transit.
  • Add Courier
  • Add Transfer
  • Assign Existing Assets with Transfer
  • Update Transfer on courier pickup
  • Add Delivery
  • Assign Existing Assets with Deliver
Both the workflows above update the status of the BOM's, Purchases, Deliveries, Transferes and Assets automatically at stages are complete.

Create a Asset
Assets can be created on their own using the Add buttons.
From a Bill of Materials, Purchase or Delivery using the Create New Asset button.
Or by using Bulk Import or the API.

In addition to the different ways to start the Asset creation process the Create Asset page has the option to create indvidual Assets or Assets in bulk using a supplied quantity.

Create a Supplier
Create a new Supplier to be associate with a BOM. Not to be confused with Manufacturers, Suppliers are the vendors of the physical products your assets are purchased from.

For example, if you have a Cisco switch and are purchasing it from ACME Networking Equipment, the supplier name would be 'ACME Networking Equipment'.

Create a BOM
A BOM is created so you can associate a set of asset you require but haven't yet purchased.

BOM's can be created on their own using the Add buttons.
Or from a Purchase using the Create New Purchase button when viewing the Purchase which will automatically associate this BOM with that object if you've done things out of order.

Once a BOM has been created you can create new Assets and associate them with the BOM while viewing the BOM using the Create New Asset button.
If the assets already exist use the Assign Existing Assets button and the bulk select interface.

Create a Purchase
Purchase are used to track what you are ordering. They can be associated with multiple BOM's indepenant of a Assets own association with a BOM.

Purchases can be created on their own using the Add buttons.
Or from a BOM or Purchase using the Create New Purchase button when viewing the BOM or Purchase which will automatically associate this Purchase with that object.

Once a Purchase has been created you can create new Assets and associate them with the Purchase while viewing the Purchase using the Create New Asset button.
If the assets already exist use the Assign Existing Assets button and the bulk select interface, if you added a BOM(s) to your Purchase then the bulk select interface will be filtered to Asset associated with that BOM(s).

Create a Delivery
Deliveries are used to track what you have recieved. They can be associated with multiple Purchases indepenant of a Assets own association with a Purchase.

Deliveries can be created on their own using the Add buttons.
Or from a Purchase using the Create New Purchase button when viewing the Purchase which will automatically associate this Delivery with that object.

Once a Delivery has been created you can create new Assets and associate them with the Purchase while viewing the Purchase using the Create New Asset button.
If the assets already exist use the Assign Existing Assets button and the bulk select interface, if you added a Purchase(s) to your Delivery then the bulk select interface will be filtered to Asset associated with that Purchase(s).

It is recommended to set the Delivery's Site and Location fields to the Site and Location of where the Asset(s) are being delivered to before allocation of Assets to the Delivery takes place. When done in this ordedr Inventory Plus will update the Assets Site and Location as well.

Scanning Serial Numbers in Bulk

Serial numbers can be added to assets in bulk using the Scan button on the Assets page.

  • On the Assets page, select the assets you want to add
  • At the bottom of the page select the Scan button
  • The Bulk Scan form has a single textarea, add serial numbers one line per serial number
  • Why you apply the changes the serial numbers are assigned to the selected objects
You can enter serial numbers by hand or with a scanner if your serial numbers are on barcodes. The form validates your entries and will tell you when enough have been entered, this is to help you when scanning large numbers of serial numbers to detect when a mistake may have been made.

Deploying Assets

Netbox Devices and Modules can be created from Assets by selecting any unallocated asset and selecting Create Device button, if the device is already allocated the button will be grey'd out.

If there is an existing Device or Module you can associate that device with the Edit Assignementbutton
